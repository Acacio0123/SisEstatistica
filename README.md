## Trabalho Prosseguiu da seguinte forma:
<p>
Criacao de funcoes para dividir a complexidade do mesmo e redistribuir as tarefas, tal que:
</p>

* <strong>A funcao soma:</strong>
faz varedura de todos os valores da lista, soma e retorna o resultado da soma.
* <strong>A funcao Media:</strong>
Faz varedura dos valores da lista e calcula a media
* <strong>A funcao obterMaior:</strong>
Usa a funcao nativa <b>maximum</b> para determinar e retornar o maior valor da lista
* <strong>A funcao obterMenor:</strong>
Usa a funcao nativa <b>minimum</b> para determinar e retornar o menor valor da lista
* <strong>A funcao contarElementos:</strong>
Usa a funcao nativa <b>length</b> para determinar e retornar o tamanho da lista


# Saida de dados
<p>No fim, inicializa-se a funcao principal main IO para a execucao da saida de dados pelo monitor. 

Nesta etapa, realiza-se o seguinte:
</p>

* <strong>Estrutura condicional <b>if else</b></strong>:
esta estrutura verifica a  principio se existem valores na lista, se nao existirem valores, retorna um erro, se existirem valores, avanca para execucao do bloco else
* o bloco do else chama as funcoes a cima criadas e ilustra todas elas passando a principio a lista como argumento valido da funcao se existir pelo menos um valor na lista.
